Programming language used: JavaScript
Requirements to run this project: node.js, phaser.js, vite
External services: N/A
Additional hardware: N/A

To run: do 'npm run dev' in the terminal to run on localhost:5173, this runs the project tree in the 'client' folder which is the working / testing solution
Do 'npm run start' in the terminal to run on localhost:3000, this runs the project tree in the 'distribution' folder which is the final / official build
